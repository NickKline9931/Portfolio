My Work

React Art Gallery

Uses Art Institute of Chicago API to bring public domain artworks to the user. You can search using the search bar or by clicking a tag. Works can be saved to favorites so you can come back to them at any time.

React Resume Builder

Resume Builder that allows user to edit personal info and add education and work experience. Education and work items can be deleted and edited. Uses React for displaying forms when needed, and hiding when not needed.

React Shopping Cart

Shopping cart made with Reactjs. Allows users to choose quantity of items, while calculating price. Tracks number of items in cart.

React Weather App

Uses openweathermap API to retrieve weather data as well as coordinates, which are then sent to ipgeolocation API to retrieve timezone data. Input any city in the world to display date, time, temperature, feels like, and weather description with corresponding graphic.

Battleship

Battleship game that uses interactive grids and timed events.Tracks turns, shots, and succesful hits. Announces winner.

Sketch Pad

Uses javascript for many events. User can change size of pad, toggle pencil on and off, change pencil color, and clear board.

Sign Up Form Validation

Uses Javascript to validate fields on sign up form. Form cannot be submitted until all fields are filled out and valid. Errors are shown to the user so they can be corrected. Checks for matching password confirmation and includes drop down menu for country.

Slideshow

Contains a collection of images that automatically cycle on a timer. User can skip to next and previous image, as well as select a specific image to show. Timer is reset every time image changes, giving the user 5 seconds to view before it changes again.

Library

User can enter the title, author, and number of pages of their favorite books and store them in a library. Books can be marked as "read" or "unread" to keep track. Books can also be removed from the library with the x buttons.
